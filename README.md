
### This project fetches API data from Vegargerðin and Veðurstofa Íslands to provide current observation information from weather-stations in Iceland. 
### It fetched API data from yr.no(temporarily) for forecasts up to +9 days via coordinates.
Currently fixed on one route, Reykjavík ↔ Ísafjörður (due to limitations - school project)
